# KubeVoIP Kamailio runtime

This repo contains the Dockerfile for the Kamailio image used by KubeVoIP.

Image:

```text
ghcr.io/kubevoip/kubevoip-kamailio
```

The platform chart pins the tested tag. Most users should install KubeVoIP from
`kubevoip/kubevoip` instead of choosing component tags directly.
